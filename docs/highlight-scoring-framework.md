# Highlight Scoring Framework

## Scoring definitions

- `Context clarity`: Can the clip be understood quickly?
- `Payoff strength`: Does something meaningful happen?
- `Visual energy`: Is the moment visually active or expressive?
- `Hook potential`: Can the clip start strongly?
- `Platform fit`: Does it suit Shorts/Reels/TikTok behavior?

## Selection rule

- Keep clips scoring `18+` for fast testing.
- Review clips scoring `14-17` if they can be improved by captions or script rewrite.
- Discard clips scoring `13 or below` unless they support a larger story sequence.
