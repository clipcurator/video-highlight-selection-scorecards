# 视频高光筛选评分卡

视频高光筛选，是决定长视频里哪些瞬间值得被做成切片、Shorts、剧情浓缩或社交内容。本仓库整理体育、游戏、访谈、播客、Webinar、教程和剧情类视频的高光筛选评分框架与检查清单。

目标是在正式剪辑前，把“哪些片段最值得剪”这件事做得更稳定。

## 什么样的高光值得剪

一个强高光，通常至少具备下面三项：

- 有明确的情绪或信息回报
- 不依赖太多缺失上下文也能理解
- 有足够强的画面或语言瞬间
- 具备分享价值：冲突、惊讶或清晰 takeaway
- 能自然形成开头问题或结尾悬念

## 评分卡类型

| 评分卡 | 适合素材 | 重点 |
|---|---|---|
| 情绪波峰评分卡 | 剧情、纪录片、vlog | 惊讶、冲突、情绪反转 |
| 观点切片评分卡 | 播客、教程、Webinar | 清晰度、实用性、独立价值 |
| 动作高光评分卡 | 体育、游戏、直播 | 画面强度、节奏、回报 |
| 社交短片评分卡 | Reels、Shorts、TikTok | 钩子、速度、可分享性 |

## 评估流程

1. 先看完或概括原视频。
2. 标出候选时间戳。
3. 针对清晰度、回报、视觉强度和平台适配度打分。
4. 去掉依赖过多上下文的片段。
5. 优先保留更适合做开头的片段。
6. 把最强片段转成剪辑 brief。
